# Headshot Image

## Required File
Place your headshot image at: `public/headshot.jpg`

## Specifications
- **Format**: JPG, PNG, or WebP
- **Dimensions**: Square aspect ratio (recommended: 400x400px or higher)
- **Size**: Under 200KB for optimal performance
- **Quality**: High-resolution for crisp display on retina screens

## Fallback
If no headshot image is provided, the About component will automatically display a placeholder with your first name.

## Usage
The headshot is displayed in the About section with:
- Square container (responsive sizing)
- Animated rotating conic gradient border
- Smooth theme transitions
- Availability badge overlay
