# About Section Images

## Required Images

To complete the About section, you need to save the following images to this directory:

### 1. Glass Bust (glass-bust.png or glass-bust.svg)
- The purple glass human bust/head image
- Should be saved as: `glass-bust.png` or keep the existing `glass-bust.svg`
- Recommended size: 400x400px or larger
- Format: PNG with transparency or SVG

### 2. Glass Crystal (glass-crystal.png or glass-crystal.svg)
- The purple glass crystal/shard image
- Should be saved as: `glass-crystal.png` or keep the existing `glass-crystal.svg`
- Recommended size: 400x500px or larger
- Format: PNG with transparency or SVG

## Current Status

Currently using placeholder SVG files. Replace them with the actual images from your design for the best visual result.

## How to Replace

1. Save your glass bust image as `glass-bust.png` in this folder
2. Save your glass crystal image as `glass-crystal.png` in this folder
3. Update the file extensions in `src/components/sections/AboutNew.jsx` if needed:
   - Change `.svg` to `.png` in the image src attributes
